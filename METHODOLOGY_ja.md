# RiskLens 信用格付け手法

このドキュメントでは、RiskLens が内部信用格付けとリスクスコアを生成するために使用する定量的フレームワークについて説明します。この手法は、古典的な多変量倒産予測（**Altman Z-Score モデル**）と、最新の継続的な融資後コベナンツ監視とを橋渡しします。

---

## 1. 主要なスコアリング アーキテクチャ: Altman Z-Score

RiskLens は、上場企業のコア アルゴリズム エンジンとして、**Altman Z-Score** に大きく依存しています。Z-Score は、企業の財務健全性の 5 つの異なる側面を 1 つの予測スコアに合成する多変量数式です。

### Z-Score の計算式
`Z = 1.2(X1) + 1.4(X2) + 3.3(X3) + 0.6(X4) + 1.0(X5)`

ここで:
- **X1 (運転資本 / 総資産)**: 短期的な流動性とバッファを評価します。
- **X2 (利益剰余金 / 総資産)**: 経時的な累積収益性を測定し、設立間もない企業や慢性的に不採算な企業にペナルティを与えます。
- **X3 (EBIT / 総資産)**: 税金とレバレッジの影響から切り離された、営業効率と資産生産性の純粋な指標です。
- **X4 (株式の時価総額 / 総負債)**: 市場の自己資本を導入し、帳簿上の負債に対する市場の支払能力の認識を評価します。
- **X5 (売上高 / 総資産)**: ビジネスモデルの総資産回転率と資本集約度を測定します。

### Z-Score ゾーンの分類
生（未補正）の Z-Score は、3 つの異なるリスク トランシェに変換されます。
1. **Safe 区間 (Z ≥ 2.99)**: 強固な財務の健全性を意味します。2 年以内の倒産確率は統計的にごくわずかです。
2. **Grey 区間 (1.81 ≤ Z < 2.99)**: 中程度のリスクがあることを示します。監視の強化と与信の厳格化が求められます。
3. **Distress 区間 (Z < 1.81)**: 24 か月以内のデフォルトまたは大規模な事業再編の可能性が高い状態です。

## 2. 予想格付けマッピング スケール

生の Z-Score を、標準的な信用方針と互換性のある機関投資家レベルの言語に変換するために、RiskLens は各ゾーンを S&P 相当の予想格付けにマッピングします。

| Z-Score の範囲 | ゾーン区間 | 予想格付け | リスクカテゴリー | 説明 |
| :--- | :--- | :--- | :--- | :--- |
| ≥ 4.50 | Safe | **AAA / AA** | プライム | 債務を履行する能力が極めて高い。 |
| 2.99 - 4.49 | Safe | **A** | アッパー・ミディアム | 債務を履行する能力は高いが、悪環境の変化に影響を受けやすい。 |
| 2.50 - 2.98 | Grey | **BBB** | ロワー・ミディアム | 債務を履行する能力は適切だが、保護のパラメータが中程度。 |
| 1.81 - 2.49 | Grey | **BB** | 非投資格付け的 | 短期的には脆弱性は低いものの、不確実性に直面している。 |
| 1.20 - 1.80 | Distress | **B**| 投機的 | ビジネスや金融の悪条件に対して脆弱である。 |
| < 1.20 | Distress | **CCC / D**| 質的な実質的リスク | 現在脆弱であり、好条件に依存している。 |

## 3. 融資後コベナンツ監視

Z-Score はある時点でのデフォルト確率を提供しますが、コベナンツは継続的な「早期警戒システム」として機能します。RiskLens はこれらの指標を積極的に監視し、テクニカル デフォルトが顕在化する前にアラートを送信します。

### 標準的な監視コベナンツ:
1. **レバレッジ テスト**: `有利子負債 / EBITDA < [しきい値]` — 借り手が本業または買収によって過剰なレバレッジをかけるのを防ぎます。
2. **カバレッジ テスト**: `EBITDA / 支払利息 > [しきい値]` — 債務返済費用を賄うのに十分な現金収益が確保されているか確認します。
3. **流動性 テスト**: `流動資産 / 流動負債 > [しきい値]` — 目前の負債に対する十分な短期資産のカバー範囲を確保します。

*注: RiskLens は、融資後の監視において `null` または欠落した値を保守的に扱い、静かにコベナンツを合格させるのではなく、引受担当者による手動確認を強制するために自動的に BREACH（違反）の想定を引き起こします。*

## 4. 財務データの正規化

RiskLens は、 スコアリング エンジンに正確にデータを供給するために、国境を越えた厳密な自己組織化データ クリーニングを実行します。
- **米国 / グローバル モデル (yFinance)**: 標準的な SEC の XBRL タグを RiskLens の主要変数に調和させます。
- **中国 A 株 (AKShare)**: 中国本土の会計基準（GAAP）の分類体系（例: `長期借款`（長期借入金） を Long Term Debt に、`営業総収入`（総売上高） を Total Revenue に）を同一の数学的構造に動的にマッピングし、地理的な出身に関係なく格付けが完全に比較可能であることを保証します。
