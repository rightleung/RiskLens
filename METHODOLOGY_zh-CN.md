# RiskLens 信用评级方法论

本文档概述了 RiskLens 生成内部信用评级和风险评分所使用的量化框架。该方法论将经典的多元破产预测模型（**Altman Z-Score 模型**）与现代持续的贷后契约监测相结合。

---

## 1. 核心评分架构：Altman Z-Score

RiskLens 高度依赖 **Altman Z-Score** 作为公众公司的核心算法引擎。Z-Score 是一种多元公式，它将企业财务健康的五个不同维度综合成一个单一的预测分数。

### Z-Score 计算公式
`Z = 1.2(X1) + 1.4(X2) + 3.3(X3) + 0.6(X4) + 1.0(X5)`

其中：
- **X1 (营运资金 / 总资产)**：评估短期流动性和财务缓冲能力。
- **X2 (留存收益 / 总资产)**：衡量随时间推移的累积盈利能力，对新成立或长期亏损的企业予以低分惩罚。
- **X3 (EBIT / 总资产)**：隔离了税收和杠杆效应，是对营运效率和资产生产率的纯粹衡量指标。
- **X4 (权益市值 / 总负债)**：引入市场资本化，以评估市场对企业偿付能力相对于其账面债务的看法。
- **X5 (销售额 / 总资产)**：衡量商业模式的整体资产周转率和资本密集度。

### Z-Score 风险区间划分
未经调整的 Z-Score 被转换为三个不同的风险层级：
1. **安全区 (Z ≥ 2.99)**：意味着强健的财务状况；2 年内破产的可能性在统计上可以忽略不计。
2. **灰色区 (1.81 ≤ Z < 2.99)**：表明处于中度违约风险之中；需要加强监测并可能需要收紧信贷额度。
3. **违约危机区 (Z < 1.81)**：在 24 个月内发生违约或大规模债务重组的可能性极高。

## 2. 隐含机构评级映射表

为了将原始的 Z-Score 转换为与标准信贷政策兼容的机构性话语体系，RiskLens 将上述风险区间映射到了标普 (S&P) 等效的隐含评级上。

| Z-Score 范围 | 风险区间 | 隐含评级 | 风险类别 | 描述 |
| :--- | :--- | :--- | :--- | :--- |
| ≥ 4.50 | Safe | **AAA / AA** | 极优级 (Prime) | 履行财务承诺的能力极强。 |
| 2.99 - 4.49 | Safe | **A** | 中上级 | 履约能力强，但容易受到不利经济环境改变的影响。 |
| 2.50 - 2.98 | Grey | **BBB** | 中下级 | 履约能力勉强充足；保护参数一般。 |
| 1.81 - 2.49 | Grey | **BB** | 投机级 | 短期内较不脆弱，但面临长期的不确定性。 |
| 1.20 - 1.80 | Distress | **B**| 高度投机级| 极易受到不利商业和经济状况的影响。 |
| < 1.20 | Distress | **CCC / D**| 实质违约风险 | 违约风险极高，偿债高度依赖于极其有利的宏观环境。 |

## 3. 贷后契约监测体系

Z-Score 提供的是某一时点下的横截面违约概率，而财务契约则作为持续的“早期预警系统”。RiskLens 会主动监测这些关键指标，以便在技术性违约成为既定事实前触发警报。

### 标准监测的财务契约：
1. **杠杆测试 (Leverage Test)**：`总负债 / EBITDA < [阈值]` — 防止借款人通过内生增长或过度并购进而导致负债率过高。
2. **覆盖率测试 (Coverage Test)**：`EBITDA / 利息费用 > [阈值]` — 确保企业保留足够的支持经营所需的现金，用于支付偿债成本。
3. **流动性测试 (Liquidity Test)**：`流动资产 / 流动负债 > [阈值]` — 确保对即将到期的短期负债有足够的资产覆盖范围。

*注：在贷后监测中，RiskLens 对 `null`（空值）或缺失值的处理极为保守，系统不会默认通过这些契约测试，而是会自动假设为触发“违约 (BREACH)”，从而强制要求信贷审批人员进行人工复核。*

## 4. 全球财务数据本体规范化

RiskLens 在不同国家和监管准则的主体之间执行极其严格的数据清理与对齐，以此为评分引擎提供准确的弹药：
- **美国 / 全球模型 (yFinance)**：将标准的 SEC XBRL 标签统一整合到 RiskLens 的第一主变量中。
- **中国 A 股及港股 (AKShare)**：动态地将军团级别的中国大陆 GAAP 报表科目分类（例如，*“长期借款”* 对应 Long Term Debt，*“营业总收入”* 对应 Total Revenue）精准映射到相同的底层数学数据结构中。这保证了即便是跨越国界的各类评级，在 RiskLens 面前依然能维持绝对的公允和可比性。
